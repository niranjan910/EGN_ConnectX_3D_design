# EGN CONNECT X 2026 — 3D Floor Plan

An interactive **3D conversion** of the tentative 2D floor plan for **EGN CONNECT X 2026**, the Education Growth Network expo.

- **Event:** EGN CONNECT X 2026
- **Venue:** Bombay Convention & Exhibition Centre (BEC, NESCO), Mumbai, India
- **Dates:** November 03–04, 2026

## What's in here

| File | Description |
|------|-------------|
| `Floor_Planner_version2.html` | **Latest** — the polished, presentation-ready 3D floor plan. A single, self-contained offline HTML file; just open it in any modern browser. |
| `Floor_Planner_version1.html` | The original first-pass 3D conversion (kept for reference). |
| `EGN Connect X Floor Plan - KEZ.pdf` | The original 2D source floor plan. |
| `brain.md` | Design analysis of the floor plan (zones, booth sizes, visitor flow, recommendations). |

## The 3D floor plan

Open `Floor_Planner_version2.html` in a browser — no internet connection or install required
(Three.js is embedded inside the file).

**Contents modelled from the source plan**
- All **264 booths** across zones A–E plus the M meeting rooms, at their correct sizes (9 – 54 SQM)
- **4 conference halls** (K12, International Schools / Career Counsellors, Early Learnings, Higher Education)
- **2 entry/exit gates** and **Coffee Lounge 2**

**Interactive features**
- Realistic expo shell-scheme stands (carpet, walls, zone-coloured fascia, furniture)
- Drag to orbit · scroll to zoom · right-drag to pan · double-click to reset
- **Search** any booth by ID (e.g. `B42`) — the camera flies to it
- **Click** a booth for a details card (zone, area, footprint)
- **Zone filter** chips to isolate a single zone
- **3D / Top** view toggle, plus toggles for auto-rotate, labels, people and roof structure
